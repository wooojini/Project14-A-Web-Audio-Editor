## 📢프로젝트 소개
🎶 **AUDI(AUdio eDItor)** 🎶는 클라이언트 기반 웹 음성 편집기 서비스입니다!

언제 어디서나 **웹 브라우저와 네트워크 연결**만 가능하다면 오디오 및 음성 파형 편집을 시작할 수 있습니다!

[✨지금 바로 편집하러 가보세요!✨](http://www.audi-editor.ml/)
